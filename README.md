# CAS3106 Cheating Paper Generator
This cheating sheet generator will
• ease your hard work of pasting our course PPT pages onto the paper.
• make it easy to put more information on limited A4 pages.
You won't have to paste images into pdf editor one by one, you just put images in one folder and
run the generate and you will get quite decent cheating sheet.

## How To Use
Put images you want to put in the paper to images folder first,
And install required python packages using 'pip install reportlab pillow'
And run 'python3 main.py', and that's all!

## Environment
Python : 3.12.3
